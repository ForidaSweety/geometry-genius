1. What is DOM? what is the purpose of DOM?

Ans: In javascript it is an interface that we can manipulate the content of a website. When we use DOM the browser create a representation of website document.Using DOM we can change the style,structure also can create elements and attributes.

3.What is event bubble?

Ans: Event is in javascript is a change in the state of an object.Event bubbling happens when an element receives an event ,that event bubbles up to its parent(it can be transmitted or propagated).
